# Mi Mercado Global - Backend DynamoDB

Avance del proyecto de Bases de Datos No Relacionales.

## Integrantes
- Miguel Olaya - Luis Mateus 

## Arquitectura
- **Motor:** DynamoDB Local (Docker)
- **Infraestructura:** Docker Compose
- **Patrones de Acceso:** Usuarios, Pedidos y Detalles de Pedidos (Single Table Design).

## Cómo ejecutar
1. Tener Docker instalado.
2. Ejecutar `docker compose up -d`.
